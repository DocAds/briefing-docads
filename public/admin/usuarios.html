Usuários admin

Membros da equipe DocAds com acesso ao painel

ℹ Como adicionar um novo usuário admin:
1. No Supabase Auth → Users, crie o usuário com e-mail e senha.
2. Depois rode no SQL Editor: 

insert into public.admins (id, full_name, role)
select id, 'Nome Completo', 'admin'
from auth.users where email = 'novo@docads.com.br';
